Was macht die App?

Ein schlankes Web-Interface zur Erfassung täglicher Lehrlings-Fahrten. Daten (Punkte, Distanzen, Fahrer) kommen per JSON von einem Google Apps Script (GAS) und werden dort gespeichert.

Bedienung

  • Route wählen: Route 1/2 in der linken Spalte umschalten.
  • Reihenfolge & Auswahl: Punkte per Drag&Drop sortieren, mit Checkbox aktivieren/deaktivieren. Die Gesamtdistanz aktualisiert sich automatisch.
  • Fahrer/Datum/Schicht: links oben auswählen. Das Datum ist automatisch heute.
  • Speichern: Button Speichern sendet die Daten an den GAS-Server. Eine Bestätigung erscheint links im Infofeld.
  • Darstellung: Hell/Dunkel-Modus über den Mond-/Sonnen-Button.

Technik

  • Frontend: HTML/CSS/Vanilla JS
  • Backend: Google Apps Script (Web-App Endpoint)
  • Hauptdateien: api.js, function.js, main.js
